连接到 GitHub

每天的自动抓取由 GitHub Actions 完成，不需要在这里做任何设置。
但如果你想直接在这个网页上「编辑关键词 / 抓取来源 / 追踪进度」并保存回仓库，需要一个有权限的 Personal Access Token—— 去 GitHub → Settings → Developer settings → Fine-grained tokens 新建一个，权限只勾选这一个仓库的 Contents: Read and write，其余都不要勾。Token 只会保存在你这台电脑浏览器的本地存储里，只会直接发给 api.github.com，不会经过任何其他服务器。不想用也完全可以——你随时可以直接在 GitHub 网页上手动改 data/*.json 文件。

抓取关键词

每天的爬虫会用这些关键词去匹配各个资助来源页面上的标题和段落。跟着 TactiCall 的进展调整这里—— 比如换了新的目标市场、拿到新专利方向、或者开始考虑某个新场景时，把对应的词加进来。

抓取来源

爬虫每天会挨个打开这些网址，抓取标题+段落文字，和上面的关键词做匹配。政府/慈善机构的网站通常不需要 JS 就能读到内容， 抓取效果比较好；有些机构官网是前端渲染的（比如某些企业官网），抓取效果会打折扣——这属于正常情况，抓不到时就手动去看看。

改动只有点了「保存到 GitHub」才会生效；GitHub Actions 会在下一次每日运行时读取新的关键词/来源。
